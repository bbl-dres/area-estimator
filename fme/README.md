# FME Volume Estimator

`Volume Estimator FME.fmw` is an FME workbench that performs the same building volume estimation as the Python pipeline (Steps 1-3), using swissALTI3D and swissSURFACE3D elevation models.

> **Note:** FME requires a commercial license from [Safe Software](https://fme.safe.com/).
> The Python version in [`python/`](../python/) provides the same functionality without any license requirement.
